Cách tạo sprite sheet cá Nemo

96 frame (8 hướng × 12 frame animation) — bạn muốn tạo bằng cách nào?

A

Code tạo bằng Canvas (programmatic pixel art)

Viết code JavaScript/Canvas vẽ cá Nemo pixel-by-pixel, tự động sinh 96 frame với rotation + animation vây/đuôi. Hoàn toàn tự động, miễn phí, có thể tùy chỉnh.

Pros

  • 100% tự động, không cần tool bên ngoài
  • Dễ chỉnh màu, kích thước, animation
  • Tạo đúng format sprite data cần thiết

Cons

  • Chất lượng pixel art phụ thuộc vào code
  • Có thể trông "thô" hơn vẽ tay
B

Tải sprite miễn phí + chỉnh sửa

Tìm sprite cá Nemo có sẵn trên itch.io/OpenGameArt, rồi viết script convert sang đúng format 8 hướng × 12 frame.

Pros

  • Chất lượng art tốt (do artist vẽ)
  • Nhiều lựa chọn style

Cons

  • Khó tìm đúng 8 hướng + animation
  • Phải xử lý license
  • Cần chỉnh sửa thủ công
C

SVG → Pixel Art (hybrid)

Code vẽ cá Nemo bằng SVG vector, rồi render ra pixel art PNG ở kích thước nhỏ. Kết hợp sự mượt mà của vector với look pixel art.

Pros

  • Dễ xoay 8 hướng (vector rotation)
  • Trông smooth hơn pure pixel art
  • Dễ animate vây/đuôi bằng transform

Cons

  • Có thể mất "pixel art feel"
  • Phức tạp hơn option A
